Authentication

Doetoez API menggunakan JWT (JSON Web Token) untuk autentikasi.

Get API Key 
# Register untuk mendapatkan API key
curl -X POST "https://asistantku.privhandi.my.id/api/register" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your-email@example.com",
    "password": "your-password"
  }'
Authentication Header 
// Include API key in headers
const headers = {
  'Authorization': 'Bearer YOUR_API_KEY',
  'Content-Type': 'application/json'
};

Chat Completion API

Endpoint untuk chat dengan Doetoez AI assistant.

POST
/api/v1/chat/completions
JavaScript Example 
const response = await fetch('https://asistantku.privhandi.my.id/api/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: "doetoez-gpt-4",
    messages: [
      {
        role: "user",
        content: "Halo Doetoez, bisa bantu saya dengan kode React?"
      }
    ],
    stream: false,
    memory: true
  })
});

const data = await response.json();
console.log(data.choices[0].message.content);

Parameters

Parameter Type Required Description
model string Yes Model AI yang digunakan
messages array Yes Array pesan dalam konversasi
stream boolean No Enable streaming response
memory boolean No Gunakan memori konversasi

ChatGPT API Integration

Doetoez mendukung integrasi langsung dengan OpenAI ChatGPT.

ChatGPT via Doetoez 
// Use ChatGPT through Doetoez proxy
const response = await fetch('https://asistantku.privhandi.my.id/api/v1/proxy/chatgpt', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: "gpt-4",
    messages: [
      {
        role: "user", 
        content: "Your question here"
      }
    ],
    temperature: 0.7
  })
});
